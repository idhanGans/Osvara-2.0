## 🚀 QUICK START (3 Steps)

### Step 1: Install (5 min)

```bash
cd frontend && npm install
cd ../backend && npm install
```

### Step 2: Configure (2 min)

```
Add Stripe keys to backend/.env
```

### Step 3: Run (1 min)

```bash
npm run dev  # Both servers auto-start
# Visit http://localhost:3000
```
